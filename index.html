Regras

  • Você pode usar a linguagem que quiser.

  • Não é permitido a utilização de bibliotecas (exceto a classe Scanner do Java e o stdio.h do C/C++).

  • A submissão do seu código tem que ser antes do encerramento do desafio.

  • Nos próximos Code Golfs não será permitido a utilização de códigos anteriores.

  • O vencedor é a pessoa que desenvolver um código que cumpra o desafio e tenha a menor quantidade de bytes.

  • Em caso de empate, quem cumprir o desafio com mais rapidez leva vantagem no ranking.

  • É terminantemente proibido o uso de palavras de baixo calão.

  • É terminantemente proibida a postagem de material pornográfico.
